# Shard

Динамичная аркадная игра на реакцию, разработанная на движке Godot 4.3. Исходный код проекта является открытым и распространяется под лицензией MIT.

## Игровой процесс

Игрок зафиксирован в центре экрана. С четырех сторон (сверху, снизу, слева и справа) по направлению к центру летят стрелки. Задача игрока — вовремя реагировать на приближающиеся угрозы и отбивать их, нажимая соответствующие клавиши управления.

## Управление

Для отбивания стрелок используется стандартная раскладка:
* W — отбить стрелку сверху
* A — отбить стрелку слева
* S — отбить стрелку снизу
* D — отбить стрелку справа

## Основные особенности

* Динамичный игровой процесс, завязанный на координацию и скорость реакции.
* Полная поддержка кастомных уровней, позволяющая пользователям создавать собственные паттерны атак.
* Оптимизированная структура проекта для движка Godot 4.3.

## Кастомные уровни

Игра поддерживает загрузку пользовательских уровней. Файлы уровней должны располагаться в системной директории приложения.

### Путь к папке с уровнями (для Windows):
%appdata%\Godot\app_userdata\shard\levels\

### Инструкция по добавлению:
1. Нажмите комбинацию клавиш Win + R на клавиатуре.
2. Введите указанный выше путь в открывшемся окне и нажмите Enter.
3. Перенесите файлы ваших кастомных уровней в открывшуюся папку.
4. Запустите игру.

## Развертывание и запуск исходного кода

Для запуска и модификации проекта вам понадобятся:
1. Среда разработки Godot Engine версии 4.3.
2. Склонированный репозиторий с исходным кодом.

### Инструкция:
1. Скачайте или склонируйте данный репозиторий на свой компьютер.
2. Откройте Godot Engine и нажмите кнопку "Import" (Импортировать).
3. Укажите путь к файлу project.godot в папке репозитория.
4. Проект готов к запуску и редактированию внутри движка.
