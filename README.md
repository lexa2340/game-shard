# Shard

Динамичная аркадная игра на реакцию, разработанная на движке Godot 4.3. Исходный код проекта является открытым и распространяется под лицензией MIT.

## Игровой процесс

Игрок зафиксирован в центре экрана. С четырех сторон (сверху, снизу, слева и справа) по направлению к центру летят стрелки. Задача игрока — вовремя реагировать на приближающиеся угрозы и отбивать их, нажимая соответствующие клавиши управления.

## Управление

Для отбивания стрелок используется стандартная раскладка:
* W — отбить стрелку сверху
* A — отбить стрелку слева
* S — отбить стрелку снизу
* D — отбить стрелку справа

## Основные особенности

* Динамичный игровой процесс, завязанный на координацию и скорость реакции.
* Полная поддержка кастомных уровней, позволяющая пользователям создавать собственные паттерны атак.
* Оптимизированная структура проекта для движка Godot 4.3.

## Инструкция по созданию кастомных уровней

Вы можете создать собственный уровень под любую музыку с помощью видеоредактора OpenShot и официального конвертера, который прикреплен к релизу игры.

### Шаг 1. Расстановка меток в OpenShot
1. Скачайте и установите бесплатный видеоредактор OpenShot.
2. Импортируйте туда ваш музыкальный трек и перетащите его на таймлайн.
3. Запустите воспроизведение и в такт музыке нажимайте клавишу **M** на клавиатуре. Каждое нажатие создаст зеленую метку (маркер). Это будущие стрелки в игре.
4. Сохраните проект OpenShot. Вы получите файл с расширением `.osp`.

### Шаг 2. Конвертация проекта в код игры
1. Скачайте архив `OpenShotMarkers_Tool.zip` из раздела **Releases** этого репозитория и распакуйте все файлы в одну папку.
2. Запустите файл `OpenShotMarkers.exe`.
3. Введите полный путь к вашему файлу проекта OpenShot (например, `C:\Games\my_song.osp`).
4. Введите путь, куда сохранить готовый файл с кодом и расширением `.cs` (например, `C:\Games\level.cs`).
5. Программа автоматически прочитает ваши маркеры и создаст готовый C# класс со встроенными массивами `Timings` и `Sequence`.

### Шаг 3. Редактирование и установка уровня
1. Откройте созданный файл `level.cs` в любом текстовом редакторе (Блокнот, Notepad++).
2. По желанию измените настройки вверху файла:
   * `Name` — название вашего уровня.
   * `ArrowSpeed` — скорость полета стрелок (по умолчанию 600f).
3. В массиве `Sequence` все стрелки по умолчанию имеют значение `"up"`. Вручную замените нужные элементы на другие направления: `"down"`, `"left"`, `"right"`. Количество элементов в массиве стрелок строго совпадает с таймингами, что защищает игру от багов.
4. Нажмите комбинацию клавиш `Win + R` на клавиатуре, введите путь `%appdata%\Godot\app_userdata\shard\levels\` и нажмите Enter.
5. Создайте в открывшейся директории новую папку с названием вашего уровня и перенесите туда ваш отредактированный файл `level.cs`.
6. (Необязательно) В эту же папку вы можете положить файлы `ico.png` (иконка уровня в меню) и `vid.ogv` (фоновое видео в формате Ogg Video). Если их нет, игра запустится без них.
7. Запустите игру и выберите ваш уровень в меню.

## Развертывание и запуск исходного кода

Для запуска и модификации проекта вам понадобятся:
1. Среда разработки Godot Engine версии 4.3.
2. Склонированный репозиторий с исходным кодом.

### Инструкция:
1. Скачайте или склонируйте данный репозиторий на свой компьютер.
2. Откройте Godot Engine и нажмите кнопку "Import" (Импортировать).
3. Укажите путь к файлу project.godot в папке репозитория.
4. Проект готов к запуску и редактированию внутри движка.
